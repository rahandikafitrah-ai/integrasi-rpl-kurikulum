Aplikasi ini untuk membantu asesor RPL
Manual Penggunaan Program

	Buka aplikasi dengan klik dua kali file : IK-PRPL.html
	Isikan data Kurikulum pada Menu RPS GENERATOR berupa : Nama dan kode mata kuliah, CPL, CPMK dan SUB-CPMK
	Isikan rincian RPS Mata Kuliah dengaan mengklik tombol EDIT MODE berupa : Deskripsi MK, Referensi, Pilihan CPMK, SUB-CPMK (ketikan), BAHAN KAJIAM (MATERI), METODE, INDIKATOR PENILAIAN dan BOBOT (ketikan angka)
	Simpan Input 
	Pengeditan data RPS dapat dilakukan dengan memilih MK yang meu di edit pada menu Daftar Mata Kuliah
	Lakukan penilaian RPL dengan mengklik menu RPL Asesment (Penilaian RPL)
	Inputkan data calon mahasiswa dan pilih MASUKKAN KE DAFTAR MAHASISWA
	Lakukan Penilaian Transfer SKS jika calpn Mahasiswa transfer dengan menginput Nama Mata kuliah asal, Pilih SKS, Pilih Nilai awal, Pilih setara dengan Mata Kuliah ynag dituju, pilih nilai akhir dan simpan, setelah disimpan dapat dilakukan pengeditan kembali
	Lakukan penilaian REKOGNISI KOMPETENSI dengan pemilih mata kuliah yang ajan diklai, ketik komentar asesor, ketik bukti, pilih nilai dan simpan penilaian
	Hasil dua penilaian dapat di lihatdi sub menu KESIMPULAN dan di PRINT
	Msing masing penilaian dapat di print
	Simpan Penilaian, jika ingin membuka dan mengedit kembali Hasil Penilaian masuk ke sub menu BUKA PENILAIAN
